jgo.tools.compiler.interm.expr

ConstExpr

sealed trait ConstExpr extends Expr

An expression whose value is determined at compile-time. The public interface of the expr package permits the direct creation of untyped constants, but not of typed constants. All typed constants are derived from untyped constants at some point.
